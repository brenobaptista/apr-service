# APR Calculator

> APR calculator exercise

## Getting Started

### Essencial scripts

```sh
1. Generate the .env file:
$ cp .env.example .env

2. Install dependencies:
$ yarn

3. Build the project:
$ yarn build

4. Start the API:
$ yarn start
```

### Development scripts

```sh
Start the API with automatic restarting:
$ yarn dev

Run tests:
$ yarn test

Check tests coverage:
$ yarn test:coverage

Lint files:
$ yarn lint

Prettier files:
$ yarn prettier

Update dependencies:
$ yarn dependencies
```

The script `prepare` is required by Husky to automatically have Git hooks enabled after install. You don't need to run it manually.
